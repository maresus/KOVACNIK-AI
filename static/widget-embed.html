Kovačnik Chat Widget

Namestitev

Dodajte naslednjo kodo pred zaključni </body> tag na vaši spletni strani:

<!-- Kovačnik Chat Widget -->
<script src="https://kovacnik-ai-production.up.railway.app/static/widget.js"></script>

Obnašanje

  • Desktop (širina > 768px): Widget se avtomatsko odpre po 2 sekundah v desnem spodnjem kotu
  • Mobilni (širina ≤ 768px): Prikazana je samo ikonica - klik odpre fullscreen chat

Prilagoditve

Za prilagoditev barv, besedil ali obnašanja kontaktirajte razvijalca.

Opomba: Widget uporablja localStorage za shranjevanje session ID-ja, kar omogoča kontinuiran pogovor tudi po osvežitvi strani.

WordPress namestitev

V WordPress dodajte widget v Videz → Pripomočki → HTML blok v footer, ali pa dodajte v functions.php:

// Kovačnik Chat Widget
add_action('wp_footer', function() {
    echo '<script src="https://kovacnik-ai-production.up.railway.app/static/widget.js"></script>';
});
